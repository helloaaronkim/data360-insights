정기 실행 배치 SQL로 RFM 스코어·구매 빈도·카테고리 친화도를 계산하고, 결과를 Unified Individual에 부착하는 Data 360 핵심 기능입니다. 定期実行バッチ SQL で RFM スコア・購買頻度・カテゴリ親和度を算出し、結果を Unified Individual に付与する Data 360 の中核機能です。

💡 Streaming Insights는 실시간 이벤트 평가, Calculated Insights는 정기 배치 계산 — 두 기능은 상호 보완적입니다. Streaming Insights はリアルタイムイベント評価、Calculated Insights は定期バッチ計算 — 2つの機能は相互補完的です。

Calculated vs Streaming Insights

Streaming Insights는 이벤트 발생 즉시 실시간 평가, Calculated Insights는 스케줄에 따라 배치 SQL로 복잡한 집계를 처리합니다. Streaming Insights はイベント発生直後にリアルタイム評価、Calculated Insights はスケジュール実行のバッチ SQL で複雑な集計を処理します。

🗄️
DMO (Data Model Object)

Calculated Insights는 unified_individual_dlm, sales_order_v1 등 표준 DMO를 직접 참조하여 SQL을 작성합니다. Calculated Insights は unified_individual_dlm・sales_order_v1 など標準 DMO を直接参照して SQL を記述します。

🔢
NTILE / DATEDIFF 함수 NTILE / DATEDIFF 関数

NTILE(10)으로 고객을 10분위로 분류하고 DATEDIFF로 날짜 차이를 계산합니다. Data 360의 Calculated Insights SQL에서 사용 가능한 표준 함수입니다. NTILE(10) で顧客を10分位に分類し DATEDIFF で日数差を計算します。Data 360 の Calculated Insights SQL で利用可能な標準関数です。
