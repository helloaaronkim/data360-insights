핵심 개념 主要概念

🌊
Data Stream
데이터 스트림データストリーム
외부 소스와 Data 360을 연결하는 구성 단위. 커넥터, 소스 오브젝트, 필드, 새로고침 스케줄 등을 정의합니다. 外部ソースと Data 360 を接続する設定単位。コネクタ・ソースオブジェクト・フィールド・更新スケジュールなどを定義します。
🗃️
Data Lake Object (DLO)
데이터 레이크 오브젝트データレイクオブジェクト
Data Stream이 수집한 원본 데이터를 저장하는 레이어. 소스 스키마가 그대로 보존되어 이후 분석·세그멘테이션에 활용됩니다. データストリームが収集した元データを格納するレイヤー。ソーススキーマがそのまま保持され、以降の分析・セグメンテーションに活用されます。
🔄
Refresh Mode
새로고침 모드更新モード
Streaming: 변경 즉시 동기화. Hourly/Daily: 스케줄 배치. 데이터 유형과 실시간성 요구에 따라 선택합니다. Streaming：変更を即時同期。Hourly/Daily：スケジュールバッチ。データ種別とリアルタイム要件に応じて選択します。
🏢
Data Space
데이터 스페이스データスペース
Data 360 내의 데이터 격리 공간. 조직 내 여러 비즈니스 유닛이 데이터를 독립적으로 관리할 수 있습니다. Data 360 内のデータ分離空間。組織内の複数のビジネスユニットがデータを独立して管理できます。
