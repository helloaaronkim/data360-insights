핵심 개념 참조 コア概念リファレンス
CORE CONCEPT
Identity Resolution
아이덴티티 레졸루션 ID 解決
여러 소스 시스템의 레코드를 동일인물로 인식하고 Unified Individual을 생성·유지하는 Data 360 핵심 프로세스 複数のソースシステムのレコードを同一人物と認識し、Unified Individual を生成・維持する Data 360 のコアプロセス
MATCHING
Matching Rule
매칭 룰 照合ルール
레코드 간 일치 여부를 판단하는 규칙. 이메일 완전 일치, 이름+전화 퍼지 일치 등으로 구성 レコード間の一致判定に使うルール。メール完全一致、氏名+電話ファジー一致などで構成
OUTPUT
Unified Individual
통합 개인 統合個人
ID 해석을 통해 생성된 단일 고객 레코드. 모든 소스 시스템의 데이터가 이 오브젝트로 수렴 ID 解決によって生成される単一顧客レコード。全ソースのデータがこのオブジェクトに集約
CROSS-REF
Party Identification
파티 식별 パーティ識別
각 소스 시스템의 고유 ID를 저장하는 오브젝트. Unified Individual과 소스 레코드를 연결하는 교차 참조 테이블 各ソースシステムの固有 ID を保持するオブジェクト。Unified Individual とソースレコードをつなぐ相互参照テーブル
