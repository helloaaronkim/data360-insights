DMO란 무엇인가? DMO とは？

Data Model Object의 약자로, Data 360의 데이터를 저장하는 표준화된 오브젝트입니다. Salesforce가 정의한 스키마를 따르며 소스 데이터가 DMO에 매핑됩니다. Data Model Object の略で、Data 360 のデータを格納する標準化オブジェクトです。Salesforce が定義したスキーマに従い、ソースデータが DMO にマッピングされます。

🎯
Unified Individual이 중심인 이유 Unified Individual が中心の理由

모든 DMO는 결국 Unified Individual을 중심으로 연결됩니다. 고객의 구매·인터랙션·연락처 데이터가 모두 Unified Individual ID를 통해 통합됩니다. すべての DMO は最終的に Unified Individual を中心に結びつきます。購買・インタラクション・連絡先データがすべて Unified Individual ID を通じて統合されます。

🔧
커스텀 DMO カスタム DMO

표준 DMO 외에도 비즈니스 요구에 맞는 커스텀 DMO를 생성할 수 있습니다. 커스텀 DMO도 Unified Individual과 관계를 맺어 Calculated Insights에서 활용됩니다. 標準 DMO のほか、ビジネス要件に合わせたカスタム DMO を作成できます。カスタム DMO も Unified Individual とリレーションを持ち、Calculated Insights で活用できます。
